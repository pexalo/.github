# Pexalo

Pexalo is an AI agentic studio based in Kuala Lumpur, Malaysia. We audit business workflows before designing connected automations, specialist AI agents, and multi-agent workforces.

Our approach is structured around four stages:

**Audit → Recommend → Build → Monitor**

We focus on measurable operational outcomes, human oversight, controlled system access, and AI that works with the tools businesses already use.

## Engineering Principles

- Audit the workflow before choosing the technology.
- Use deterministic automation where rules are sufficient.
- Give every AI agent a bounded role and limited permissions.
- Keep people responsible for important decisions.
- Evaluate outputs using workflow-specific measures.
- Monitor reliability, exceptions, cost, and business outcomes.
- Connect with existing systems where appropriate.
